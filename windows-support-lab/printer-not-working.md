# Printer Not Working

## Issue
User cannot print from Windows 10 workstation.

## Initial Checks
- Confirmed printer is powered on.
- Checked USB / Network connection.
- Verified printer is set as default.

## Troubleshooting Steps
1. Restarted Print Spooler service.
2. Removed and reinstalled printer drivers.
3. Cleared print queue.
4. Tested printing from another user profile.

## Resolution
Corrupted driver package was causing issue.

## Prevention
Avoid abrupt shutdowns during printing jobs.
